Sample WebRTC STUN Server Setup

  1. Ensure the computer running the server and the Magic Leap device are on the same wireless network.
  2. Determine the IP address of the computer that will be used to run the server, on the command line use ifconfig on Mac or Linux or ipconfig on Windows to find it.
  3. Start the signalling server by running python server.py (provided in the server folder with this sample).
  4. Enter the server IP address and click on the Connect button.
  5. Accept the request for the browser to access the camera and microphone.
  6. A video feed from the computer's camera will be visible in video panel on the left.
  7. Once a peer is connected, its video will be shown in the video panel on the right.
  8. Once a peer is connected, the chat box will be activated. All incoming and outgoing messages will be displayed in this box. Use the input text field and Send button to send messages to the connected peer.
  9. Use the "Send Random Binary Data" button to send a list of 5 randomly generated integers to the peer as binary data through the data channel.
  10. This client is set to disconnect itself from the server and stop the local streams when the peer disconnects.

For more information about WebRTC, see Getting Started With WebRTC.
